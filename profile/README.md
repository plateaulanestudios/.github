# Plateau Lane Studios

Software, electronics, and fabrication — built on Whidbey Island, WA.

We are an eight-person partnership forming a cooperative. Each partner operates through their own LLC. We build things together and share the infrastructure to do it well.

**What we build:**

Most firms specialize. We work across software, electronics, and physical fabrication because the problems we find most interesting don't respect those boundaries — and neither do we.

- Production software for industrial operations — cattle processing, facility monitoring, business management
- Custom electronics — RF welders, CNC controllers, specialized hardware
- Physical fabrication — automotive builds, home design and construction
- Client web services — static sites, WordPress hosting, deployment automation

**How we work:**
- Small teams, deep craft, no unnecessary complexity
- We take on a small number of projects each year where the fit is right
- Each project has an owner. The co-op provides the framework.
