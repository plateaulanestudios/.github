# Plateau Lane Studios

Software, electronics, and fabrication — built on Whidbey Island, WA.

We are an eight-person partnership forming a cooperative. Each partner operates through their own LLC. We build things together and share the infrastructure to do it well.

**What we build:**
- Production software for industrial operations
- Custom electronics and fabrication tools
- Client web services and hosting

**How we work:**
- Small teams, deep craft, no shortcuts
- Each project has an owner. The co-op provides the framework.
- We choose our clients as much as they choose us.
